# Script Audit Checklist — Shameless Application

**Summary**: Reusable Shameless audit checklist for ad scripts and hook sets — fiber-first positioning, hook diversity (Q1 2026 patterns), Golden Script structure, length, information gap, compliance, voice/style, flavor names, stat accuracy. Includes a worked example. Brand-neutral checklist lives in [[script-audit-checklist]].

**Sources**: fiber-first-positioning.md, hook-framework.md, script-leaderboard.md, creative-strategy.md, compliance-guardrails.md, shameless-information-gaps.md, brand-voice.md, product-facts.md.

**Last updated**: 2026-04-17

---

Use when a user submits a script or hook set and asks "check if it can be improved," "audit this," or variants. The checklist walks the script against every relevant Shameless wiki page in the right order.

## How to Use

1. Read the script and hook set in full before checking anything
2. Walk the checklist top to bottom
3. Flag findings by category with a direct wiki citation
4. Offer a revised draft only after the audit is delivered
5. If the user asks for the revised draft to be filed, create a new wiki page under the appropriate category

## The Checklist

### 1. Positioning & angle

- **Fiber-first?** Does the script lead with fiber or route sub-angles through fiber as the mechanism? See [[fiber-first-positioning]].
- **26g hero stat present?** Is the daily-dose framing anchored to the number?
- **Sub-angle routing** — if the script is about weight, GLP-1, keto, or blood sugar, is fiber still named as the *why*?
- **Not leading with fiber?** Acceptable only for retargeting, urgency/scarcity statics, or deep-skeptic taste cold traffic ([[fiber-first-positioning#When NOT to Lead with Fiber]]).

### 2. Hooks

- **Pattern diversity** — are the hooks all Stat Shock? Per [[hook-framework#q1-2026-hook-pattern-taxonomy-8-patterns]], Stat Shock hits 42.6% hook rate but only 7.1% hold rate. Set should include at least one:
  - **Demographic / Condition Callout** (#1 pattern — 57% hook / 22.7% hold / 1.51x ROAS): perimenopause, 40+, "the girlies," keto, anti-supplement
  - **Confession / Subverted Expectation** (best CVR at 3.6% — selective but converts)
  - **Information gap** (mechanism threat — see §7)
- **Specificity** — is the hook for a *specific person in a specific situation*, or generic "every American"?
- **First-word rule** — hook opens with a verb, strong noun, or number. Not "So today I wanted to..." See [[hook-framework#four-execution-components]].
- **Avoid "ad speak"** — no "Hack," "Cheat Code," "Must-Have." Replace with native reactions: "OMG," "Wait," "Confession:" See [[hook-framework#what-consistently-fails-as-a-hook]].
- **Untested inventory** — if recycling proven hooks, consider adding one from [[hook-framework#high-potential-untested-hooks]].

### 3. Script structure (Golden Script elements)

Per [[script-leaderboard#winning-structure-constant-across-top-scripts]] and [[shameless-creative-strategy#the-golden-script-formula-q4-2025]], top scripts include all of:

- [ ] Personal / emotional / stat hook
- [ ] Problem or tension expansion
- [ ] Product reveal (in-narrative, not an announcement)
- [ ] Stat delivery: **70 cal / 3g sugar / 3g net carbs / 26g fiber** (all four)
- [ ] Mechanism or differentiator (fiber, dietary compatibility)
- [ ] Social proof (reviews, Amazon rank, friend rec, 250K+ bags sold)
- [ ] "**Eat the whole bag**" permission reframe — the universal winning line (~$67 CPA vs. $116 account avg)
- [ ] CTA (URL + tap/click)

Missing 3, 4, or 7 correlates with lower ROAS in the top-spend set.

### 4. Length

Cold-traffic: 60s+ wins ($67.25 CPA / 1.03 ROAS) per [[shameless-creative-strategy#what-winning-ads-have-in-common]]. Don't over-compress to 30s on cold UGC.

### 5. Information gap (hold-rate fix)

If the hook is Stat Shock, the body *must* have a mechanism beat or the 6.69% hold-rate problem repeats. See [[shameless-information-gaps]] for the five Shameless-specific mechanisms:

1. Mucosal degradation — gut eating itself
2. Butyrate / colon cells starving
3. Serotonin / gut-brain axis
4. Blood sugar rollercoaster
5. Gut diversity collapse

Rule of thumb: 15–20 seconds of mechanism tension before the 26g stat delivers relief.

### 6. Compliance

Check against [[compliance-guardrails]]:

- **Hard stops** — drug names (Ozempic, Wegovy, semaglutide, etc.), "melt fat," disease/cures claims, diabetic safety, cholesterol claims, body-transformation imagery
- **Borderline outcome language** — "fixed me," "cured my gut," "transformed my body" — soften to personal/experiential ("my body remembered what it was supposed to do," "bathroom situation was back to normal")
- **Permitted now** — weight-loss narrative, "natural appetite suppressant" (paired with fiber), craving control, satiety — all with fiber as mechanism

### 7. Voice & style

Per [[brand-voice]] and saved user feedback:

- No em dashes anywhere
- Sounds human, not AI — no formulaic openers, no perfect parallelism
- Visceral hooks, failure sequences before product enters, believable discovery moments
- Single-flavor CTAs (not a variety pack list)
- Emotionally powerful close

### 8. Flavor names

Always use the exact SKU name. "OMG Peach" not "Sour Peach." "Sour Strawberry Splash" not "Strawberry." No shortening, no paraphrasing.

### 9. Stat accuracy

Cross-check against [[product-facts]]:

- 26g fiber / bag
- 3g sugar / bag
- 3g net carbs / bag
- 70 calories / bag
- Keto-friendly, vegan, gluten-free

Don't restate stats from older scripts without checking — the Snack Smart VSL had a legacy "3 to 8g net carbs" line that's now factually wrong.

## Worked Example 1 — Fiber script based on GLP-1 constipation template (2026-04-17)

**Brief**: User asked for a fiber-focused script based on a proven GLP-1 constipation script, with GLP-1 removed.

**Audit findings:**

| Category | Finding | Fix |
|---|---|---|
| Hooks | All 5 drafts were Stat Shock / Problem-recognition (42.6% hook, 7.1% hold, 1.01x ROAS) — no Demographic/Condition Callout (#1 pattern at 57% / 22.7% / 1.51x ROAS) | Added perimenopause, "the girlies" TMI, lazy-wellness, myth-busting, info-gap mechanism hooks |
| Hooks | Generic audience ("every American," "average person," "your stomach") — not a specific person | Anchored each revised hook to a named ICP or situation |
| Hooks | Hook #4 "fix took three days" borderline outcome claim | Softened to "my body remembered what it was supposed to do" |
| Script | Missing information gap — direct cause of 6.69% hold-rate problem | Added butyrate/colon-cells-starving mechanism beat (15s of tension before stat) |
| Script | Missing mechanism explanation (fiber → regularity → relief) | Mechanism now precedes product reveal |
| Script | Missing social proof element | Added "250K+ bags sold, 4.5 stars on Amazon" beat |
| Script | Missing "Eat the whole bag" permission reframe — the universal winning line | Integrated: "Ate the whole bag that night, guilt-free" |
| Script | Missing 3g net carbs stat (only had 3 of 4 canonical stats) | Added to stat block |
| Script | Length ~30s — below 60s+ cold-traffic benchmark | Body expanded with mechanism + social proof to ~55-60s |
| Compliance | "Fixed my gut in three days" was outcome-adjacent | Softened to "the gut stuff sorted itself out" |
| Voice | No em dashes ✓, single-flavor CTA ✓ (OMG Peach), relatable-skeptic tone ✓ | No changes needed |
| Flavor | Used "OMG Peach" ✓ (not "Sour Peach") | No changes needed |

**Highest-upside change**: Adding the butyrate mechanism beat. Directly targets the hold-rate gap flagged in [[fiber-first-positioning#q1-2026-performance-validation]].

## When to File vs. When to Answer Inline

- **File as wiki page** when the audit is substantial, covers multiple scripts, or creates a reusable pattern
- **Answer inline only** when the fix is a single-line tweak

If filing, add to [[index]] under "Playbooks" and append an entry to [[log]].
